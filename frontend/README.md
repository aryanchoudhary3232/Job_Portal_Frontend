# Nexus Talent Frontend

Next.js frontend for the job portal with:

- marketing landing page
- login and registration
- student dashboard
- recruiter dashboard
- staff operations console

## Run

1. Copy `.env.example` to `.env.local`.
2. Run `npm install`.
3. Run `npm run dev`.

The frontend expects the backend API gateway at `NEXT_PUBLIC_API_BASE_URL`.
